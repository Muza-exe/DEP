# Air Paint (рисование жестами без мыши)

Программа захватывает движение руки через камеру и позволяет рисовать в воздухе:
- **указательный палец** — рисование;
- **указательный + средний** — режим выбора (цвет и толщина кисти в верхней панели);
- **Eraser** — стирание (рисует белым);
- **`c`** — очистить холст;
- **`q`** — выход.

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install opencv-python mediapipe numpy
```

## Запуск

```bash
python3 hand_paint.py
```

## Требования

- Python 3.9+
- Веб-камера
